# Lost & Sound
> 우연히 들은 노래의 단편적인 기억으로 곡을 찾아주는 웹 서비스

## 무엇을 하는 프로젝트인가
일상에서 노래 한 구절을 듣거나 친구가 소셜 네트워크 서비스에 공유한 앨범의 사진을 보고 어떤 앨범인지 궁금한 적이 많았으나, 이러한 궁금증을 해결하는 데 어려움이 있어 기획하게 되었습니다. 
일상에서 우연히 들은 노래의 단편적인 기억(가사 일부, 앨범 커버의 이미지 등)만으로 해당 곡을 찾아주고, 취향에 맞는 새로운 음악까지 연결해 주는 웹 서비스입니다.

* **대상 사용자:** 길거리, 카페 등에서 들은 노래의 제목을 몰라 답답했던 경험이 있는 사람, 새로운 음악을 발굴(Digging)하고 싶은 사람.

## 사용 방법
1. 웹사이트에 접속하여 검색창에 기억나는 노래 제목이나 가수를 입력합니다.
2. '가사/제목으로 검색' 버튼을 누릅니다.
3. 하단에 나타나는 앨범 커버, 곡명, 아티스트 정보를 확인하고 30초 미리듣기를 재생합니다.

## 입력 / 출력
| PRD에 적은 기능 | 실제로? |
| :--- | :--- |
| 가사/제목으로 검색하면 결과 리스트가 나온다. | 됨 |
| 빈 칸으로 보내면 안내 메시지가 뜬다. | 됨 |
| 이상한 글자(ㄱㄴㄷㄹ)를 넣으면 결과 없음 안내가 뜬다. | 됨 |
| 이미지(앨범 커버) 업로드로 검색한다. | 안됨 (기능 준비 중 경고창 뜸) |

## 아직 안 되는 것 (부숴보며 찾은 것)
* 이미지(앨범 커버)를 활용한 검색 기능은 현재 버튼 UI만 구현되어 있으며 실제 Vision API 연동이 필요합니다.
* Apple iTunes Search API를 임시로 사용하여, 기획했던 '추천 트랙 5~10개 제공' 대신 단일 곡 검색 및 30초 미리듣기만 제공됩니다.
